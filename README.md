# 张雪峰 · 游戏合集

同一组主角 —— **张雪峰 / 雪碧 / 巧乐兹** 与复用的原站背景音乐，做成 **五款玩法和美术风格都完全不同** 的网页小游戏。打开根目录 `index.html` 即是选择大厅。

灵感来源：`https://ottomate.games/zxf`（原版是「张雪峰快跑」横版跑酷）。本项目五款均为**全新设计**，与原版玩法不同；仅 BGM（`bgm.m4a`）与失败音效（`death.mp3`）按要求直接复用。

## 五款游戏

| 目录 | 名称 | 玩法 | 美术风格 |
|------|------|------|----------|
| `game/`  | 透心凉大作战 | 滑动挥刀切水果（雪碧/巧乐兹/通知书，躲炸弹），连击 + FEVER | 明亮夏日 **卡通描边** |
| `game2/` | 透心凉叠叠乐 | 一拍即合 **叠塔**，错位削边，完美对齐连击 | **16-bit 像素** 复古街机 + CRT |
| `game3/` | 霓虹弹弹弹 | 拖板 **打砖块**，多球/慢放/轰炸道具，逐关加速 | **霓虹赛博朋克** synthwave |
| `game4/` | 神点拨 | **点醒**迷茫同学，金句**连锁**扩散，守住老师的耐心（原创玩法） | **黑板粉笔** 手绘涂鸦 |
| `game5/` | 弹射名师 | 拉弓**弹射**张老师撞墙反弹，一杆**连撞**一片，躲天坑（原创玩法） | **剪纸手工** 拼贴 |

每款都有：加载页 / 标题 / 游玩 / 暂停 / 结算 全流程，移动端优先（触屏 + 安全区 + 禁缩放），鼠标/键盘同样可玩，画面有粒子、震屏、连击、音效（WebAudio 合成）等打磨。

## 本地预览
```bash
# 在仓库根目录
python -m http.server 8080
# 打开 http://localhost:8080/   选择任意一款
```

## 部署到 GitHub Pages
仓库已含自动部署流水线 `.github/workflows/deploy.yml`（监听 `main`/`master`）。已推送到远端后，只需：
1. 打开仓库 **Settings → Pages → Build and deployment → Source** 选 **GitHub Actions**。
   （若仓库为私有，GitHub Pages 需要 Pro，或把仓库改为 Public。）
2. 等 **Actions** 里的 “Deploy to GitHub Pages” 跑完（约 1 分钟）。
3. 访问 `https://<用户名>.github.io/<仓库名>/` 即是游戏大厅。

之后每次 `git push` 会自动重新部署。所有资源均为相对路径，子路径下也能正常加载。

> 调试：任意一款的地址后加 `?dev=1` 可启用 `window.__zxfDev` 测试钩子，正常游玩无副作用。
